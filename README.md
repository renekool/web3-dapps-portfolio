## 🛠️ General Tech Stack Summary

- **Smart Contracts:** Solidity (EVM-compatible), Foundry, Hardhat.
- **Frontend / Client Layer:** Next.js (React 18 & 19), Tailwind CSS, Framer Motion, Vanilla CSS.
- **Web3 Integrations:** Ethers.js v6, Viem, Wagmi, EIP-712 Meta-transactions, IPFS storage.
- **Testing & Quality Assurance:** Foundry unit testing, Playwright E2E testing, Hardhat unit testing.

## 📖 How to Run

Each project is fully containerized or organized with a clear separation of concerns between smart contracts (`sc` or `contracts` directory) and client-facing web applications (`web` or `dapp` or `frontend` directory). Refer to the specific execution guides inside each subdirectory for comprehensive, step-by-step setup instructions.
